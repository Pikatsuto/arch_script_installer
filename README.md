### Description
Arch Scripte Installer is a clean and complete archlinux installation script integrating at choice :
- Karnel: Linux, LTS, ZEN, TKG-PDS
- AUR
- Chaotic-AUR
- Powerpill
- Bash, Fish
- Xfce, gnome
- Docklike Taskbar for XFCE
- Gnome tweaks
- Desktop Arch Update Indicator
- Wine, Wine-TKG-PDS
- Gamemode
- WayDroid
